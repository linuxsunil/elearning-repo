# eLearning Standards & Templates System

Three documentation files working together:

## 📋 CLAUDE.md
Project standards for eLearning repositories
- 60+ learning theories with applications
- SCORM/xAPI compliance requirements
- WCAG 2.1 AA accessibility standards
- Storyboard template (per ADDIE)
- QA & PR review protocol

## 🎯 LESSON.md
Lesson design standards & templates
- 7 lesson types (Knowledge, Skill, Scenario, etc.)
- Problem-Based Learning structure
- 8 interaction templates (click-reveal, branching, etc.)
- Accessibility checklist per interaction
- Lesson-level QA checklist

## 👤 ID_SKILL.md
Professional expertise documentation
- 10+ years instructional design experience
- ADDIE + SAM hybrid methodology
- 60+ learning theories applied
- Use cases & client outcomes
- Engagement models

## 🚀 How They Work Together
